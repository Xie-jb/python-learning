# Day 9：字符串处理和正则表达式

## 今日目标

能从文本中提取手机号、邮箱、订单号等有用信息。

## 知识点

- `split()`：按规则切割字符串
- `replace()`：替换字符串内容
- `strip()`：去除首尾空格
- `in`：判断是否包含某段文本
- `re` 模块：正则表达式
- `re.findall()`：提取所有匹配内容

## 常见用法

```python
import re

text = "联系人：张三，电话：13812345678"
phones = re.findall(r"1[3-9]\d{9}", text)
print(phones)
```
