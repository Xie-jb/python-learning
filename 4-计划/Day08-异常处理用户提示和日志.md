# Day 8：异常处理、用户提示和日志

## 今日目标

让脚本出错时不直接崩溃，而是给用户清楚提示。

## 知识点

- `try / except`：捕获错误
- `ValueError`：输入格式错误
- `FileNotFoundError`：文件不存在
- `Exception`：通用错误
- `logging`：记录日志
- 友好的错误提示

## 常见用法

```python
try:
    age = int(input("请输入年龄："))
except ValueError:
    print("年龄必须是数字")
```
