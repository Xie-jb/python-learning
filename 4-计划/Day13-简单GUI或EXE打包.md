# Day 13：简单GUI或EXE打包

## 今日目标

让普通用户更容易使用你的工具。

## 二选一路线

### 路线A：简单窗口界面

适合给不会用命令行的客户。

学习：

- `tkinter`
- 按钮
- 输入框
- 文件选择
- 弹窗提示

### 路线B：打包成EXE

适合交付 Windows 用户。

学习：

- `pyinstaller`
- 单文件打包
- 交付目录整理
- 常见打包问题

## 常见用法

```bash
pyinstaller -F your_script.py
```

## 练习任务

1. 给一个工具做简单窗口，或把一个脚本打包成 EXE。
2. 准备一个交付文件夹。
3. 放入程序、示例文件、使用说明。
4. 找一个新文件测试工具是否还能正常运行。

## 任务目标

- 能把脚本变得更适合交付
- 能整理交付文件夹
- 能写清楚使用步骤

## 当天交付

- `day13_tool_gui.py` 或打包后的 EXE
- `交付说明.txt`
- 示例文件

## 接单关联

闲鱼客户大多不懂 Python。能给 EXE 或简单窗口，成交率会更高。
