# Day 15：作品集、闲鱼接单准备和报价模板

## 今日目标

整理作品，准备开始接简单订单。

## 需要整理的作品

至少整理 3 个作品：

- 学生或客户信息管理小系统
- Excel 订单整理工具
- 批量文件整理助手

## 每个作品需要包含

- 功能说明
- 使用方法
- 示例输入文件
- 示例输出结果
- 常见问题
- 可定制点

## 闲鱼接单话术

```text
你好，我可以做 Python 自动化脚本，比如 Excel 整理、批量文件改名、数据筛选、文本提取、简单网页公开数据采集。

你可以发一下：
1. 原始文件示例
2. 想要的最终效果
3. 是否需要打包成 exe
4. 是否有固定格式要求

我看完后给你确认价格和时间。
```

## 报价参考

- 简单 Excel 筛选、合并：30-80 元
- 批量改名、文件整理：30-100 元
- 文本提取手机号、邮箱：50-150 元
- 简单网页公开数据采集：100-300 元
- 带 GUI 或 EXE 打包：额外加 50-150 元

## 需求确认模板

```text
为了确认需求，我需要看一下：
1. 原始文件示例
2. 你想要的最终结果示例
3. 数据量大概多少
4. 是否需要以后重复使用
5. 是否需要打包成 exe
```

## 不建议接的订单

- 绕过验证码
- 抢购秒杀
- 登录他人账号
- 采集隐私数据
- 大规模采集敏感信息
- 明显违法违规用途
