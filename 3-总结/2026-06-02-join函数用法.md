# join 函数用法

#Python #字符串 #join

## 本次学习内容

`join()` 用来把列表里的多个字符串连接成一个字符串。

## 核心概念

格式：

```python
"连接符".join(字符串列表)
```

可以理解成：

```python
"用什么连接".join(要连接的内容)
```

## 示例代码

```python
lst = ["a", "b", "c"]

result = ",".join(lst)
print(result)
```

输出：

```text
a,b,c
```

意思是：用 `,` 把 `"a"`、`"b"`、`"c"` 连接起来。

## 常见写法

```python
" ".join(["hello", "world"])  # hello world
",".join(["a", "b", "c"])     # a,b,c
"".join(["py", "thon"])       # python
"\n".join(["第一行", "第二行"]) # 每个元素一行
```

## 易错点

`join()` 只能连接字符串。

```python
nums = [1, 2, 3]

result = ",".join(map(str, nums))
print(result)
```

输出：

```text
1,2,3
```

## 记忆点

一句话记住：

```python
"用什么连接".join(要连接的字符串列表)
```

`join()` 改变的是元素之间的连接方式，不会删除元素内部原本的内容。
